# Currency Converter Utility App

### React Features Used
- `useState`
- `useEffect`
- `useNavigation`
- `FlatList`

### Other methods demonstrated
- Typescript
- API fetches for currency data
  
### To get started:
   ### `npx expo install`
   ### Install dependencies
   - @react-navigation/native
   - @react-navigation/stack
   - expo-font
   ### `npm run`
   ### Scan QR code on output to view on mobile
